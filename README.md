# CorrectionJavaExercices

Ce dépôt contient les **corrections des exercices** associés au support de cours de **programmation Java** destiné aux étudiants de **BTS SIO – option SLAM**.

L'objectif est d'accompagner les étudiants dans l'apprentissage progressif du langage Java à travers des **exercices corrigés et des exemples de code**.

---

# Objectif pédagogique

Ce projet accompagne un **support de cours Java complet** permettant d'apprendre :

- les bases du langage Java
- la programmation orientée objet (POO)
- l'utilisation des API Java
- la gestion des collections et des exceptions
- la création d'interfaces graphiques avec **JavaFX**
- l'accès aux **bases de données**
- la réalisation de **petits projets Java complets**

Le cours est conçu pour une **initiation progressive au développement Java**.

---
